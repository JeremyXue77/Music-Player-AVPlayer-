# Music-Player-AVPlayer-



使用 AVPlayer 簡單實作出一個音樂播放器，：



播放器功能：

* 播放 / 暫停
* 上、下一首
* 重複播放
* 不重複播放，播放下一首歌
* 歌單選擇歌曲
